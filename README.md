# RUPEYLO Loan Form

This repository contains a minimal static loan request form. Pushing to `main` will deploy the site to GitHub Pages using the included workflow.

- Live URL after deployment: https://jmddatasheet-blip.github.io/rupyelo-loan-form/

To deploy:

```bash
git add .
git commit -m "Add site and Pages workflow"
git push origin main
```

The GitHub Actions workflow will publish the repository root to GitHub Pages. After the first successful run, the site will be available at the URL above.
